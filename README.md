# Workspace

This workspace contains multiple experiments/projects.

## AS9102 FAI Generator (PySide6 GUI)

Run:

```bash
python -m as9102_fai
```

First-time setup (Windows):

```powershell
./scripts/run_as9102_fai.cmd
```

Note: Python virtual environments (`.venv/`) are machine-specific. If you copied this repo from another computer and it already contains a `.venv/`, delete it and re-run the script.

See [as9102_fai/README.md](as9102_fai/README.md) for usage notes.
